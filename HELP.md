# Git Commands
## Version
```
git --version
```
## Help
```
git help
```
## Configurate Global User Name and Email
```
git config --global user.name "John Doe"
git config --global user.email "john.doe@example.com"
```
# Init Repository
```
git init
```
# Repository Status
```
git status
```
# Simplify Repository Status
```
git status -sb
```
# Add All Files
```
git add .
```
# Add Modified Files
```
git add -A
```
# Add Files
```
git add <file>
```
# Unstage Added Files
```
git reset <file>
```
# Commit
```
git commit -m "The commit message."
```
# Reverse All Changes
```
git checkout -- .
```
# Reverse File Changes 
```
git checkout -- <file>
```
# Log
```
git log
```
# Log in One Line
```
git log --oneline
```
# Log With All Detail
```
git log --oneline --decorate --all --graph
```
