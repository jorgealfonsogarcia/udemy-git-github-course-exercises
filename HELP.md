# Git Commands
## Version
```
git --version
```
## Help
```
git help
```
## Configurate Global User Name and Email
```
git config --global user.name "John Doe"
git config --global user.email "john.doe@example.com"
```
# Init Repository
```
git init
```
# Repository Status
```
git status
```
# Add All Files
```
git add .
```
# Add Modified Files
```
git add -A
```
# Add Files
```
git add <file>
```
# Commit
```
git commit -m "The commit message."
```
# Reverse All Changes
```
git checkout -- .
```
# Log
```
git log
```
# Unstage Added Files
```
git reset <file>
```
